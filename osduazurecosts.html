💰 Azure Cost Estimate with OSDU-Compatible Alternatives

Resource Type Azure Service Est. Monthly Cost Potential Alternatives for OSDU Notes
Storage Account Azure Storage (Hot LRS) £2–£5 ❌ None — required Used for blob storage and Terraform backend
Key Vault Azure Key Vault Standard £1–£3 ❌ None — required Stores secrets, keys, and certs
App Insights Application Insights £15–£30 Azure Monitor (basic logs) Can reduce telemetry volume or use basic logging
Log Analytics Workspace Azure Monitor Logs £2–£10 Azure Storage logging Store logs in blob storage instead
Cosmos DB Account Azure Cosmos DB (Single Region) £20–£100+ Azure Table Storage, PostgreSQL Flexible Server, MongoDB Atlas Depends on schema, query needs, and OSDU data partitioning
App Service Plan Standard S1 (1 instance) £50–£60 Azure Container Apps, Azure Functions Use serverless or containerized workloads
App Service Azure Web App Included Azure Static Web Apps, Azure Functions Depends on whether you need dynamic compute
Private DNS Zones Azure DNS Private Zones £0.40 per zone ❌ None — required Needed for internal name resolution
Virtual Network + Subnets Azure VNet Free ❌ None — required Core networking layer
Role Assignments / SPs Azure AD Free ❌ None — required Identity and access control

🧠 Notes on Alternatives

  • Cosmos DB: If you don’t need global distribution or multi-model APIs, Azure Table Storage is a cheap NoSQL alternative. For relational queries, PostgreSQL Flexible Server is OSDU-compatible and much cheaper.
  • App Service Plan: If you’re just hosting APIs or microservices, Azure Container Apps or Azure Functions can cut costs dramatically.
  • App Insights: You can reduce ingestion volume or switch to basic Azure Monitor logs to save money.
